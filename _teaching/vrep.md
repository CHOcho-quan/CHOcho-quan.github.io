Junior Year Project about a six-foot robot. We do path & motion planning via the VREP platform using simple methods such as A* and RRT.

* Github Address : [VREP Motion Planning](https://github.com/CHOcho-quan/VrepMotionPlanning)
* Junior year first semester's project. It controls a robot to do simple tasks such as moving across some obstacles by doing A* or RRT algorithms. Using easy path planning method and control method to plan the robot's path.
* I was the team leader of the project. We finish the project satisfyingly.
